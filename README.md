# Oil-Dashboard
"Interactive Excel dashboard showing Global vs Nigeria Oil Value (1990-2014) with KPIs and visual chart"
This repository contains an Excel dashboard analyzing global oil value trends with a focus on Nigeria. The dashboard provides a clear visual comparison between Nigeria’s oil value and the global oil market over the period 1990–2014.
Features
KPI Cards:
Total Global Oil Value (2014)
Nigeria Oil Value (2014)
Nigeria’s share of global oil value (%)
Interactive Line Chart:
Shows oil value trends for Global and Nigeria
Years 1990–2014 on the X-axis
Oil value in USD (Billions / Trillions) on the Y-axis
Insight Section: Short narrative highlighting key trends and comparisons
Optional Images / Icons: Subtle oil and gas graphics for better visual appeal
Clean & Professional Layout:
Gridlines hidden
Consistent color palette
Locked KPI cards to prevent accidental edits
Files
Oil-Dashboard.pdf → Static view of the dashboard
Oil-Dashboard.xlsx → Interactive Excel dashboard
Dataset Source
Kaggle: Global Oil and Natural Gas Production, Prices, Exports, and Net Exports
Years covered: 1990–2014 (oil), 1990–2014 (Nigeria), with incomplete 2014 data
